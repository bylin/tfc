First things first, let's remove NA values (which happens when there is no corresponding RNA-Seq count data for housekeeping genes) and duplicated tsRNA/miRNA rows. Duplicates arise from different loci transcribing the same gene. We want to keep only 1 locus.

DESeq2 estimates gene dispersions in a 3 step process. First, it considers individual genes, and looks at their dispersions across all samples. Second, it fits a curve to the individual gene dispersions. Third, it uses the curve as a prior to shrink the individual dispersions towards the curve. Thus, duplicated genes will affect the fitted curve, and subsequently the dispersion adjustments towards the prior.

We want to make both miRNA and tsRNA signatures, and use them to predict cancer status (normal vs tumor). First, we want to perform differential expression, tumor versus normal. So we need to split into training/testing sets. This split needs to have an appropriate number of normal and tumor samples.

randomSamplePatients <- function(metadata) {
    # sample pairs of samples first
    paired_metadata <- metadata[metadata$paired, ]
    paired_samples <- sample(as.factor(paired_metadata$participant_id), nrow(paired_metadata)/2)
    # then sample single sample patients
    unpaired_metadata <- metadata[!metadata$paired, ]
    unpaired_samples <- sample(as.factor(unpaired_metadata$participant_id), nrow(unpaired_metadata)/2)
    # combine patient ids
    c(as.character(paired_samples), as.character(unpaired_samples))
}
random_sample <- randomSamplePatients(prad_metadata)
training_counts <- feature_counts[, prad_metadata$participant_id %in% random_sample]
training_metadata <- prad_metadata[prad_metadata$participant_id %in% random_sample, ]
testing_counts <- feature_counts[, !(prad_metadata$participant_id %in% random_sample)]
testing_metadata <- prad_metadata[!(prad_metadata$participant_id %in% random_sample), ]

runDESeq2 <- function(counts, metadata) {
  colData <- data.frame(row.names = colnames(counts), patient = as.factor(metadata$participant_id), sample_type = metadata$sample_type, type = "single-read")
    # feature count column names correspond to metadata rows
  controls <- (nrow(counts) - 9):nrow(counts) # last 10 genes for PRAD are controls
  dds <- DESeqDataSetFromMatrix(countData = counts, colData = colData, design = ~ patient + sample_type)
  dds <- estimateSizeFactors(dds, controlGenes = controls)
  dds <- estimateDispersions(dds)
  dds <- nbinomWaldTest(dds)
  results(dds)
}
